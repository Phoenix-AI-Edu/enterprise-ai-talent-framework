# AI 研究助理、合規二審與地端 RAG 部署實踐報告

## 📌 背景說明
本案協助證券公司在金管會金融業 AI 指引與低手續費競爭雙重壓力下，導入地端多代理人 RAG 系統。研究部門面臨手動財報與數據處理工時瓶頸；營業員則在轉型財富管理顧問時遭遇強烈技術焦慮；資訊部與法遵部門則需確保個資不出防火牆、禁忌詞不過濾即不得外發。

## ⚠️ 核心阻力與痛點分析
1. **Research 手動工時瓶頸**：分析師每日耗時抓取公開資訊 PDF 財報與手動計算 EPS、毛利率，高重複性與高錯誤風險。
2. **營業員轉型焦慮**：被迫學習共同基金、海外債券與結構型商品，恐懼 AI 直接取代其開發客戶業務。
3. **金管會合規紅線**：零幻覺要求與個資保護不容出差錯，系統必須留下完整審計軌跡，否則面臨撤銷牌照與巨額罰款。

## 🛠️ 鳳凰顧問解決方案
*   **多代理人研究助理**：Planning Agent 追蹤除權息日程，Execution Agent 自動計算 EPS 與毛利率，Compliance Agent 掃描禁忌詞。
*   **營業員第二大腦（Agentic RAG）**：不需 Prompt 學習曲線，以語音或簡單文字查詢投資對照表，AI 極速生成合規話術與圖表。
*   **合規二審制（Double-Check）**：涉及個股投資建議，必須經合格證券分析師人工審核並手動簽章後，才可發送給客戶。
*   **地端 DMZ + DLP + Token 熔斷**：所有客戶資料 全面 不出防火牆，自動遮罩個資，單月算力預算硬熔斷防止天價帳單。

## 📈 專案成效 (ROI)
*   **Research 工時降低**：自動財報數據與 EPS 計算，釋放研究員專注高階判斷。
*   **營業員 AI 採納率提升**：AI 定位為第二大腦而非取代工具，大幅降低轉型抗拒。
*   **合規審計軌跡完整**：每份報告留下人工簽章與禁忌詞檢查紀錄，董事會與金管會稽查皆可追溯。
