# 📘 單元一：AI 基礎理論與 2026 技術演進

本單元是「鳳凰AI」企業教材的基石。我們將徹底告別傳統教科書中枯燥的「大數據 5V」與「機器學習硬核公式分類」，轉而從「範式轉移」與「2026 現代大模型營運架構」出發，引導企業決策者與研發人員建立直覺、深刻且具備商業決策價值的技術視野。

---

## 🎯 學習目標
* **理解範式轉移**：掌握從「規則導向（Rule-based）」到「特徵表示學習（Learning Representations）」的思維跨越。
* **掌握核心架構**：能向非技術團隊清晰解釋 Tokenizer、Embedding、Transformer 的自注意力機制（Self-Attention）如何協同運作。
* **洞察訓練機制**：理解「預訓練（Pre-training）」與「偏好對齊（Alignment - RLHF/DPO）」在降低企業導入幻覺與合規風險中的商業價值。
* **技術對接**：理解 2026 年大模型如何透過工具調用與 MCP（Model Context Protocol）實現從「純文字對話」到「代理執行（Agentic Action）」的質變。
