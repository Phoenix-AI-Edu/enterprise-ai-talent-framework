## 📝 核心課題模組快覽

### 1. 企業五大核心 AI 任務
1. **分類 (Classification)**：信用卡詐欺偵測、郵件分類、瑕疵品影像判斷。
2. **摘要 (Summary)**：長合約審查、財務報表提煉、會議紀錄與Todo自動生成。
3. **問答 (Q&A)**：基於企業內部 SOP 的智能客服、RAG 知識庫檢索。
4. **預測 (Prediction)**：供應鏈庫存預測、智慧能源電力負載管理。
5. **自動執行 (Agentic Action)**：跨系統自動填表、發送 Email、觸發 ERP/CRM 流程。
