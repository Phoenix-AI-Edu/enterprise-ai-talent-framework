## 📝 核心課題模組快覽

### 1. 自監督預訓練與 DPO 偏好對齊
* **數學本質**：自監督學習極限接龍的限制，RLHF 獎勵模型的繁瑣痛點。
* **DPO 損失函數**：2026 主流的直接偏好最佳化，如何用幾百筆標註數據達成品牌聲調與防護欄的快速對齊。

### 2. 多代理人系統 (Multiagent Systems)
* **設計模式**：
  - **規劃代理 (Planning Agent)**：負責大任務拆解與任務流程調度。
  - **執行代理 (Execution Agent)**：專業工具調用（Tools Calling）與子任務執行。
  - **審查代理 (Review Agent)**：品質管控、自我修正（Self-Correction）與防範法規紅線。

### 3. Model Context Protocol (MCP)
* **協定三大支柱**：標準化 Resources、Prompts 與 Tools 協定。
* **變革價值**：大模型跨系統連接的插拔革命，實現 AI 與 PostgreSQL, GitHub, Slack 等企業核心資產的無縫融合。
