# 📈 單元四：機器學習技術理論與實務案例

本單元打破傳統機器學習教材僅著重公式推導與學術分割實驗的舊思維，轉而聚焦於企業導入機器學習的 **ROI 演化路線**、**多模態特徵工程 (Feature Engineering)** 實務，以及工業界最頂尖的預測性維護與能源調度案例。

---

## 🎯 學習目標
* **精準評估 ROI**：理解傳統機器學習（ML）與生成式大模型（LLM）的邊界，學會依據計算成本與商業場景做出最優技術路徑選擇。
* **精通多模態特徵工程**：學會對時間序列、 tabular 表格數據、非結構化文本進行高級特徵工程處理與融合。
* **智慧工業實戰**：掌握半導體產線「預測性維護（PdM）」與工廠「電力尖峰負載預測」的完整機器學習落地架構與量化指標。

## 📝 核心課題模組快覽

### 1. 機器學習之企業 ROI 決策演進
* **技術對照表**：傳統 ML、深度學習與 LLM 在算力、可解釋性與數據量上的邊界。
* **ROI 決策公式**：將業務優化帶來的省下成本減去初期開發、算力與運維成本，得出最理性的技術路徑。

### 2. 多模態特徵工程
* **時間序列**：滑動窗口統計、差分特徵、週期與季節特徵。
* **結構化表格**：目標編碼與獨熱編碼、交互特徵。
* **非結構化文本**：Sentence Embedding 將歷史日誌向量化。

### 3. 工業級旗艦案例
* **預測性維護 (PdM)**：蝕刻機台故障預測、FFT 特徵工程與 SHAP 可解釋性歸因。
* **智慧電力負載預測**：融合生產排程、氣象預報與時序數據的 Temporal Fusion Transformer 預測及 EMS 能源管理對接。
