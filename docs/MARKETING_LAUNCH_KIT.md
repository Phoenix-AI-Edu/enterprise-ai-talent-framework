## 4. 一頁式簡報大綱 (One-pager Pitch)

**標題**：中央廚房 AI 營運助理 —— 從門市到總部的智慧中樞

**版塊 1：我們解決什麼問題？ (The Problem)**
- **溝通無效率**：門市依賴通訊軟體回報，格式混亂，總部需大量人工轉打資料。
- **決策有落差**：缺乏標準化審核流程，高風險事件（如設備損壞）容易被忽視或延遲處理。
- **系統整合難**：直接修改現有 ERP/POS 成本極高且具風險。

**版塊 2：我們的示範方案 (The Solution - Pilot Ready)**
- **自然語言轉工單**：支援模糊語意解析，精準分類任務。
- **總部戰情與覆核**：Human-in-the-loop 設計，敏感操作由管理員最終決策。
- **Sandbox 預覽環境**：即時生成標準化 API Payload，讓 IT 團隊「看得到、測得到」再談整合。

**版塊 3：預期商業價值 (Expected Value)**

- 建立透明、可追溯的門市異常通報與決策留存機制。
- 透過沙盒測試，顯著降低後續 IT 系統整合的試錯成本。
