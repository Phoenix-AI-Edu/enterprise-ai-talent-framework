# R2/R3 Remediation Plan

## Batch 1：Active Cases 中 Overall Risk = High 的案例
處理 cases.json 與被引用的 8 個 High Risk HTML 檔案。替換真實品牌名稱（A公司、興達扣件等）與具體 ROI 數字（30%、150% 等）。

## Batch 2：Standalone HTML 中涉及真實品牌/地名等可識別資訊
處理包含鼎盛、恆達、岡山、明茶道、悅品等地名與品牌的孤立 HTML 檔案。改為泛用型描述，不改檔名。

## Batch 3：Standalone HTML 中僅涉及未驗證數字或 ROI 宣稱
處理剩下的 HTML 檔案，將具體金額（1.9億、100萬等）或成效數字改寫為質化描述。
