# 聯發光學 (okayama_filter) 一頁式 AI 落地與合規戰略畫布 (A3 Strategy Canvas)

## 🏗️ 一、 企業背景與核心業務挑戰 (Current Status & Challenges)

1. **企業背景**  
   聯發光學篩選工業位於高雄岡山，專注於提供高頻光學影像篩選服務，主要客戶為當地大型螺絲螺帽廠。公司年營收約新台幣 1.5 億元，致力於成為南台灣扣件防線的「出貨守門員」。

2. **特許監管大限與品質紅線**  
   目前，聯發光學的 AI 影像缺陷檢測系統面臨嚴峻的品管挑戰，特別是在應對特斯拉等高端客戶的品質要求上。誤報率高達 15% 的情況已經導致顯著的客訴風險，並可能引發高額賠償。

3. **一線變革痛點**  
   - **高誤報率**：AI 系統將正常品誤判為缺陷，導致人力資源浪費及視覺疲勞。
   - **抵制情緒**：品管班長對 AI 系統持反感態度，影響了系統的有效運作及員工士氣。

## 🔒 二、 核心 AI 解決方案架構 (Target AI Architecture)

```text
+---------------------+
|   AI 檢測系統      |
|                     |
|  +---------------+  |
|  |  影像處理模組 |  |
|  +---------------+  |
|        |            |
|   +----v-----+      |
|   |  AI 模型 |      |
|   +----+-----+      |
|        |            |
|  +-----v------+     |
|  |  結果輸出  |     |
|  +-----+------+     |
|        |            |
|  +-----v------+     |
|  | HITL 校對  |     |
|  +-------------+    |
+---------------------+
```

- **部署策略**：在 AI 檢測系統中，首先進行影像處理，接著由 AI 模型進行初步篩選，最後通過 HITL 校對流程進行人機二次校對，確保準確性。
- **人機雙簽工作流**：建立 AI 判斷與人員校對的明確流程，確保每一個判斷都有人工驗證，以減少誤報和提升信心。

## 🛡️ 三、 金融合規與資安防線 (Governance & Compliance)

- **有人在環 (HITL) 決策簽章設計**：設計包含人工校對的簽章流程，確保每一個檢測結果都有專業人員的確認。
- **DLP 個資網閘**：實施數據損失防護 (DLP) 措施，確保個人資料不被未經授權的訪問。
- **API 算力/Token 財務熔斷防線**：建立 API 認證及財務熔斷機制，防止資源濫用與不當操作，保護企業資金安全。

## 👥 四、 變革管理與員工賦能 (Change Management & Enablement)

- **定位**：識別員工在變革過程中的角色，強調 AI 系統在提升效率與準確性方面的優勢。
- **激勵機制**：針對品管班長設計特別獎金制度，鼓勵其參與 AI 系統的優化與數據標記。
- **變革心理化解方案**：提供心理輔導與教育訓練，幫助員工適應新技術，減輕對取代的恐懼。
- **KPI 指標重塑**：將 AI 準確性與人員校對效率納入 KPI 指標，確保綜合考量系統效能與人力資源運用。

## 💸 五、 租稅折抵與財務 TCO 精算 (Tax Incentives & TCO)

- **研發租稅抵減 15% 實務包裝申報指引**：提供專業指導，協助企業申請研發租稅抵減，以降低財務負擔。
- **Buy vs. Build 3年期 TCO 評估**：分析自建與外購解決方案的總擁有成本 (TCO)，提供決策依據，確保資源最有效利用。
